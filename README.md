# GPX Tools

My own tools for manipulating GPX files. Written so it's easier to create proper GPX files for running based on a series of not-so-perfect GPX sources.

## Dev

Build

```
cargo build
```

Run

```
cargo run
```

Run tests

```
cargo test
```
